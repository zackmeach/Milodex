## SIL Open Font License 1.1 (summary)

The OFL grants recipients the right to:

- Use the fonts freely in documents, software, and products.
- Embed or bundle the fonts in software (including applications distributed to users).
- Modify and redistribute the fonts under the same license.

The OFL requires that:

- Font software may not be sold on its own.
- Derivative font names may not use the original reserved font names.
- The license and copyright notices must be preserved in distributions.

Full license text is in each family's `OFL.txt` file.
